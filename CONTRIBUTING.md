# Contributing Guidelines

Thank you for your interest in this project!

## Not Accepting Direct Code Contributions

This is a **personal portfolio project** and is not open for external code contributions (pull requests). However, **issues are welcome!**

The repository is public for:

- **Educational purposes** - allowing others to learn from the implementation
- **Transparency** - showcasing my work and coding practices
- **Reference** - helping others build their own portfolios
- **Community feedback** - receiving bug reports and suggestions

## What You Can Do

**Open issues** - Bug reports, questions, and suggestions are welcome  
**Fork and learn** - Feel free to fork this repository and use it as a learning resource  
**Build your own** - Use this as inspiration for your own portfolio project  
**Share feedback** - Feedback via issues or email is appreciated

## What This Repo Doesn't Accept

**Pull requests** - Will be politely closed (I prefer to implement fixes myself)  
**Feature requests** - May be considered but this is tailored to my personal needs  
**Bug reports** - Yes! Please report bugs via issues  
**Questions** - Technical questions about implementation are welcome

## How to Report Issues

When opening an issue, please include:

- **Description** - Clear description of the issue or suggestion
- **Steps to reproduce** (for bugs) - How to replicate the problem
- **Expected vs actual behavior** (for bugs)
- **Screenshots** (if applicable)
- **Browser/device info** (for UI issues)
